# Magefleet Magento2 Cloud Build

## Purpose
Dockerfile, configurations and scripts to build Magefleet Magento Infrastructure.

## Installation
Install the Composer package:
```bash
composer require magefleet/magento2-cloud-build:PHP-VERSION
```
